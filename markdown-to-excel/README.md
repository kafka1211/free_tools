# Markdown to Excel

## 概要

- ブラウザ上で動作するクライアントサイドの Markdown から Excel(.xlsx) への変換ツールです。
- JavaScript (ExcelJS) を利用して、入力された Markdown をその場で Excel ファイルに変換し、ダウンロード可能にします。

## バージョン情報

**Current Version: 1.1**

## 機能一覧 (v1.1)

以下の Markdown 記法に対応し、Excel 上で適切なスタイルに変換されます。

- **見出し (Headers)**: `#` ～ `######` に応じてフォントサイズが調整されます。
- **リスト (Lists)**:
    - 箇条書き (`-`, `*`)
    - 番号付きリスト (`1.`)
    - インデントによる階層化に対応
- **タスクリスト (Task Lists)**:
    - `[ ]` → ☐ (未完了)
    - `[x]` → ☑ (完了)
- **テーブル (Tables)**:
    - 標準的な Markdown テーブル記法に対応
    - 列の配置指定 (`:---`, `:---:`, `---:`) による左寄せ・中央寄せ・右寄せに対応
    - ヘッダー行の背景色設定
- **引用 (Blockquotes)**:
    - `>` で始まる行をグレーの斜体・左側罫線付きで表現
- **水平線 (Horizontal Rules)**:
    - `---`, `***`, `___` でセルを結合した罫線を描画
- **テキスト装飾**:
    - **太字** (`**text**`)
    - *斜体* (`*text*`)
    - ~~取り消し線~~ (`~~text~~`)
    - `インラインコード` (`` `text` ``)
    - [リンク](url) (ハイパーリンクとして機能)

## 使用方法

1. ローカル環境に `markdown-to-excel.html` を配置
2. ブラウザで `markdown-to-excel.html` を開く
3. テキストエリアに Markdown を入力
4. **「Excelに変換」** ボタンをクリック
5. 自動的に `.xlsx` ファイルのダウンロードが開始されます

### 設定
- **インデント単位**: スペース2つ、または4つから選択可能です。

## 注意点
- 大きな Markdown テキストや複雑なレイアウトの場合、ExcelJS の制限・処理時間に注意してください。
- 画像の埋め込みには現在対応していません。
